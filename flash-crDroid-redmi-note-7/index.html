1. Reference
  1. You need to have a Xiaomi account that has been given permission for unlocking device.
  2. Download MiFlash from https://www.miui.com/unlock/index.html
  3. Download the crDroid ROM for Redmi Note 7 https://crdroid.net/lavender/9
  4. Download firmware update from [1]
  5. Download NikGapps from https://sourceforge.net/projects/nikgapps/files/Releases/NikGapps-T/. Core variant is enough.
  6. Download OrangeFox recovery for Redmi Note 7 https://orangefox.download/en/device/lavender
    • crDroid version 9.8 introduce dynamic partition. We need to use OrangeFox because is the one that support dynamic partition since https://orangefox.download/release/64e67545531cf4e269b8aaa3.
  7. Download Magisk from https://github.com/topjohnwu/Magisk/releases
  8. Connect the phone to your laptop and open MiFlash to unlock your device following the instruction.
  9. Reboot into bootloader mode
  10. Execute the command to detect your device fastboot.exe devices
  11. Flash OrangeFox recovery from bootloader with fastboot.exe flash recovery recovery.img
  12. Reboot into recovery mode fastboot.exe reboot
  13. Wipe Dalvik, cache, system, vendor from recovery
  14. Copy the downloaded files to your phone in recovery mode
  15. Flash firmware update from recovery
  16. Flash crDroid from recovery
  17. Reboot into system once and reboot again into recovery
  18. Flash NikGapps from recovery
  19. Boot into system
  20. Install Magisk-XX.apk
  21. Follow this instruction to complete Magisk installation https://topjohnwu.github.io/Magisk/install.html
  22. Install F-Droid
  23. Install Aurora Store from F-Droid
  24. Install Gboard
  25. Install Chrome from Aurora Store
  26. Install 腾讯应用宝 from Chrome
