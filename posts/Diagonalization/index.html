What is diagonalization?

For a n×nn \times nn×n matrix AAA the diagonalization is the process to find a diagonal matrix DDD and a invertible n×nn \times nn×n matrix PPP that holds following equality A=PDP−1A = PDP^{-1}A=PDP−1

Why we need diagonalization?

We often need to calculate the power of the matrix, this operation AkA^{k}Ak is expensive to calculate when AAA and kkk is big enough. However, if the matrix AAA is diagonalizable. The power operation is then cheaper because Ak=(PDP−1)k=PDkP−1A^{k} = (PDP^{-1})^k = PD^{k}P^{-1}Ak=(PDP−1)k=PDkP−1 and the powers of diagonal matrix is straight forward.

When a matrix is diagonalizable

A n×nn \times nn×n matrix AAA is diagonalizable if and only if AAA has nnn linearly independent eigenvectors.

What matrices are not diagonalizable

  • A rotation matrix is not diagonalizable over real fields
  • Nonzero nilpotent matrices

How to find all the eigenvalues and corresponding eigenvectors

From the previous conclusion (A−λjI)⋅p⃗j=0⃗(A-\lambda_{j}I) \cdot \vec{p}_{j} = \vec{0}(A−λj​I)⋅p​j​=0 and invertible matrix theorem, then a number λ\lambdaλ is an eigenvalue of AAA if and only if f(λ)=det(A−λI)=0f(\lambda) = det(A-\lambda I)= 0f(λ)=det(A−λI)=0, f(λ)f(\lambda)f(λ) is called the characteristic polynomial of AAA.

The process of finding all the eigenvalues and corresponding eigenvectors is following

  1. Find all the roots λ\lambdaλ of the characteristic polynomial.
  2. For each λ\lambdaλ, compute corresponding eigenvectors by finding linearly independent non-trivial solutions of the homogeneous system of equations (A−λI)⋅p⃗=0⃗(A-\lambda I) \cdot \vec{p} = \vec{0}(A−λI)⋅p​=0

Application of diagonalization

One of important application of diagonalization is PCA. In PCA we need to calculate eigenvectors and eigenvalues of covariance matrix. Since covariance matrix is symmetric matrix hence it is always diagonalizable.
