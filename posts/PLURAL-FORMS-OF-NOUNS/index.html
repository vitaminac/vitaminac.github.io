名词的复数形式

元音结尾的名词, 复数形式结尾于-s

libro : libros (libro + s)

pluma : plumas (pluma + s)

chico : chicos (chico + s)

señora : señoras (señora + s)

定冠词(el, la) 同样要换成复数形式 "los" 和 "las".

el libro: los libros la pluma: las plumas el chico: los chicos la señora: las señoras

如果是辅音结尾则加上后缀 -es

el borrador: los borradores (borrador + es)

la universidad: las universidades (universidad + es)

el profesor: los profesores (profesor + es)

la ciudad: las ciudades (ciudad + es)

如果名词结尾与 -ión, 加上 -es和去掉重音

el avión : los aviones

la converesación : las conversaciones

la sección : las secciones

la televisión : las televisiones

你或许会奇怪为什么 "avión" 不是阴性. 它并不符合我们之前提到的规则 -ción 和 -sión 结尾的阴性名词.

如果名词结尾 -z, 加上 -es 然后把 z 改成 c

el lápiz : los lapices la voz : las voces el tapiz : los tapices la actriz : las actrices

当指多个个体不同性别的时候 使用阳性复数形式

2 perros + 6 perras = 8 perros 1 gato + 8 gatas = 9 gatos

复合名词是由两个单词合成一个

例如: abre + latas ¡ abrelatas / 开 + 罐头 = 开罐器

这些复合名词永远是阳性, 他的复数形式是把 "el" 改成 "los"

el abrelatas los abrelatas

el paraguas los paraguas

总结

  • 元音 结尾的名词 加 -s
  • 辅音 结尾的名词 加 -es
  • -z 结尾的名词 把 -z 改成 -c 然后加上 -es
  • -ión 结尾的名词 去掉重音加上 -es
  • 如果指男女混合的群体 需要使用阳性形式
  • 对于复合名词只需要把 "el" 改成 "los"
